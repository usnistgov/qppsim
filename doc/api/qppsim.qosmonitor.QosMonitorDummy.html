qppsim.qosmonitor.QosMonitorDummy module¶

Documentation¶

Module with a dummy QoS Monitor. This monitor only reports ‘NaN’ and ‘0’ as the QoS values for every bearer.

Bases: qppsim.qosmonitor.QosMonitorBase.QosMonitorBase

get_qos()[source]¶

Report dummy QoS values composed of ‘NaN’ and ‘0’ for every active bearer.

QosTracing objects are created, and if configured, they are written to the QoS trace.
