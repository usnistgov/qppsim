qppsim.preemption.PreemptionSamplePreemptSome module¶

Documentation¶

Module with a basic Pre-emption implementation.

class qppsim.preemption.PreemptionSamplePreemptSome.PreemptionSamplePreemptSome[source]¶

Bases: qppsim.preemption.PreemptionBase.PreemptionBase

Class for a sample Pre-emption implementation. This implementation selects all the bearers with a lower ARP than the bearer that needs to be accepted and pre-empts them one by one until enough resources are freed. For the QoS pre-emption, only one bearer with the lowest ARP is pre-empted.

attempt_preemption(new_bearer_id, new_bearer_arp, rbs_needed, rbs_used)[source]¶

Attempt to pre-empt bearers to free resources for a new bearer. This implementation selects all the bearers with a lower ARP than the bearer that needs to be accepted, and pre-empts them one by one until enough resources are freed.

qos_preemption(ue_imsi, bearer_id, bearer_arp)[source]¶

Attempt to pre-empt a bearer to ensure that the specified bearer can comply with its QoS contract.
