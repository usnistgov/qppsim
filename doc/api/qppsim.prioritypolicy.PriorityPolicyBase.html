qppsim.prioritypolicy.PriorityPolicyBase module¶

Documentation¶

Module with the Base class for Priority Policies. Priority Policies define how GBR, MBR, QCI, ARP, PVI, and PCI values are assigned to bearers based on the application type (name) and the UE.
