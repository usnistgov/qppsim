qppsim.Amc module¶

Documentation¶

Module with the Adaptive Modulation Coding information. It contains a constant dictionary with the TBS for 1 to 50 RBs for each value of the MCS, and a function that returns how many RBs are needed to provide a given data rate at a given MCS.

qppsim.Amc.get_rbs_for_rate(mcs, rate)[source]¶

Get the number of RBs needed to provide the requested data rate at the provided MCS
