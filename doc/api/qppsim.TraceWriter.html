qppsim.TraceWriter module¶

Documentation¶

Module with the TraceWriter, a class to compile all the tracing operations in in a single method. It also defines the Network Overhead to be used in the application traffic traces when reporting the network-level packet size.

qppsim.TraceWriter.NETWORK_OVERHEAD = 30¶

Network overhead in bytes to be used when tracing the network-level packet size. This value is taken from the ns-3 simulations, as follows:

  • 8 Bytes in UDP header
  • 20 Bytes in IPv4 header
  • 2 Bytes in PDCP header
